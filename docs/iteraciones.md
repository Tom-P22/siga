# Registro de iteraciones del prototipo SIGA

## v0.1 - MVP de acceso

- Login y cierre de sesión simulados.
- Panel inicial con acceso a módulos.
- Primera validación del flujo de funcionario.

## v0.2 - Base institucional

- Tailwind CSS v4 y paleta institucional.
- Navegación responsive para escritorio, tablet y tótem.
- Perfiles de funcionario y administrador.

## v0.3 - Control vehicular

- Captura QR y reconocimiento LPR simulados.
- Admisión temporal para vehículos.
- Reglas de 180 días para particulares y 90 para diplomáticos.
- Formato imprimible en dos copias.

## v0.4 - SAG y autoatención

- Declaración jurada digital.
- Derivación de alimentos y mascotas a inspección.
- Registro previo de pasajeros.
- Validación de autorizaciones para menores.

## v0.5 - Trazabilidad y analítica

- Persistencia en localStorage.
- Auditoría de accesos y operaciones.
- Indicadores operativos.
- Exportación CSV y reporte imprimible.

## v1.0 - Prototipo funcional

- Integración de todos los recorridos.
- Ayuda contextual y guion de demostración.
- Vista de evolución del MVP.
- Build TypeScript y lint verificados.
- Plan de pruebas y control de cambios documentados.
